# eOrth App Theme Colors

이 문서에는 eOrth 어플리케이션에서 사용되는 주요 테마 색상 및 네온 컬러 코드들이 정의되어 있습니다. 
Figma 디자인 시스템 및 Liquid Glass UI Kit를 기준으로 구현되었습니다.

---

## 1. 기본 테마 컬러 (COLORS)
앱의 백그라운드, 카드 레이아웃, 그리고 주요 텍스트와 보조 색상으로 사용되는 고유 코드들입니다.

| 변수명 | 색상 코드 / 값 | 설명 |
| :--- | :--- | :--- |
| **bg** | `#0A0A0F` | 앱 전체 백그라운드 (어두운 미드나잇 블루블랙) |
| **card** | `#2E2E3B` | 카드 및 바인더 레이아웃 배경색 |
| **divider** | `#1A1A26` | 얇은 구분선 및 슬롯 테두리색 |
| **purpleNeon** | `#BF85FC` | 주요 포인트 네온 퍼플 (선택 뱃지 등) |
| **purpleDeep** | `#6B21A8` | 깊은 보라색 (그라데이션 종단점 등) |
| **purpleBg** | `rgba(107,33,168,0.25)` | 보라색 반투명 오버레이/배경 |
| **purpleBorder** | `rgba(191,133,252,0.3)` | 보라색 반투명 테두리 |
| **purpleThumb** | `#1A0A2E` | 썸네일 어두운 보라색 배경 |
| **white** | `#FFFFFF` | 밝은 텍스트 및 기본 흰색 |
| **textDim** | `#A1A1B0` | 보조 텍스트 (회색빛 도는 중간 밝기 톤) |
| **textMuted** | `#4A4A59` | 비활성화/어두운 보조 텍스트 |
| **red** | `#FF3B30` | 에러, 삭제, 경고 및 신고용 레드 |
| **redBg** | `rgba(255,59,48,0.1)` | 경고용 반투명 배경 |
| **redBorder** | `rgba(255,59,48,0.2)` | 경고용 반투명 테두리 |

## 2. 네온 포인트 컬러 (NEON)
메탈릭 코인이나 그라데이션 광택, 네온 링(Neon Ring) 효과를 극대화할 때 활용되는 생동감 넘치는 컬러 세트입니다.

* 🩵 **cyan**: `#22D3EE` (청록 네온)
* 💙 **blue**: `#3B82F6` (파란 네온)
* 💜 **purple**: `#A855F7` (보라 네온)
* 🩷 **magenta**: `#D946EF` (자홍 네온)
* 🌸 **pink**: `#F472B6` (분홍 네온)

---

## 3. 글래스모피즘 반투명 사양 (GLASS)
리퀴드 글래스(Liquid Glass) 효과를 줄 때 유리 재질의 투명도와 반사광을 시각화하기 위해 사용되는 투명도 값입니다.

* **border** (유리 테두리 반사선): `rgba(255, 255, 255, 0.30)`
* **innerTop** (유리 내측 상단 글로우): `rgba(255, 255, 255, 0.16)`
* **innerBottom** (유리 내측 하단 투과율): `rgba(255, 255, 255, 0.02)`
* **specular** (빛 반사 하이라이트): `rgba(255, 255, 255, 0.55)`

---

## 4. 통계 칩 그라데이션 (STAT_GRADS)
프로필 화면의 통계 칩(기록 수, 팔로잉 등) 배경에 씌워지는 선명한 퍼플 그라데이션 값입니다.

* **Start Color**: `#BF85FC` (Neon Purple)
* **End Color**: `#6B21A8` (Deep Purple)
* **Code**: `['#BF85FC', '#6B21A8']`
