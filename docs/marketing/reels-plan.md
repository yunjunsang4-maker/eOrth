# eOrth 인스타그램 릴스 시리즈 기획안

> 목적: **인물 없이 앱 화면만으로** 기능을 하나씩 소개하는 홍보 릴스 시리즈
> 타깃: 한국(국내) + 글로벌(영어) 병행 → 릴스마다 한/영 자막 버전 2개 제작
> 원칙: **릴스 1개 = 기능 1개**, 15~25초, 첫 3초에 결과부터

---

## 0. 공통 제작 규격 (모든 릴스 동일)

| 항목 | 값 |
|------|------|
| 화면비 | 9:16 (1080×1920), 30fps |
| 녹화 | **실기기 화면 녹화** (아이폰: 제어센터 화면 기록 / 안드로이드: 내장 녹화). 에뮬레이터 X |
| 길이 | 15~25초 (훅 0~3초 / 데모 3~18초 / CTA 18~22초) |
| 인트로 | 0.5초 — 검은 배경(#0A0A0F)에 eOrth 네온 로고 페이드 인 |
| 아웃트로 | 1.5초 — 로고 + "eOrth" + CTA 문구(스토어/베타 안내) |
| 브랜드 색 | 배경 #0A0A0F · 보라 네온 #BF85FC · 텍스트 흐림 #A1A1B0 |
| 자막 | 다크 배경이라 **흰색/네온 텍스트**, 굵게, 화면 하단 1/3 안전영역 위 |
| 음악 | 인스타 "인기 오디오"에서 잔잔한 신스/로파이 (도달률 유리) |
| 편집 | CapCut(무료) 권장 — 자막·속도조절·전환 |

### 촬영 실전 팁
- **손가락 조작이 보이게** 천천히. 탭할 때 0.5초 멈춤 → 편집에서 확대(줌인)로 강조.
- 상태바(시간·배터리) 깔끔하게: 방해금지 모드 ON, 배터리 100% 근처.
- 한 기능당 **3~4번 반복 녹화** 후 가장 매끄러운 테이크 채택.
- 앱 로딩·빈 화면 구간은 편집에서 컷. 실제 "와" 하는 순간만 남기기.

## 1. 촬영 전 준비 — 데모 계정 & 데이터 세팅

릴스가 화려하려면 **미리 채워둔 데모 계정**이 필수입니다. 빈 지구본·빈 피드는 절대 촬영 금지.

### 데모 계정 구성 (권장)
- **핸들**: `@eorth_demo` (또는 홍보용 공식 계정)
- **방문 국가 8~12개**: 색이 골고루 퍼져 지구본이 화려하게 — 예) 일본 🇯🇵 · 태국 🇹🇭 · 미국 🇺🇸 · 프랑스 🇫🇷 · 이탈리아 🇮🇹 · 베트남 🇻🇳 · 호주 🇦🇺 · 스페인 🇪🇸
  - 대륙이 아시아/유럽/아메리카/오세아니아로 퍼지게 → **대륙별 통계 바**가 예쁘게 참
- **여행 기록 10~15개**: 형식을 골고루 섞기
  - Feed(사진형) 6개, Cut(네컷) 3개, Blog(글) 2개, Snap(즉석) 3개
  - 별점은 4~5점 위주(★★★★★가 많아야 별점 분포/궤도 예쁨)
- **사진**: 저작권 프리 여행 사진(Unsplash/Pexels) 또는 본인 사진. 밝고 색감 좋은 것 위주.
- **여행 카드 대표사진**: 각 카드 coverUri 지정 → 프로필 그리드가 사진으로 꽉 차 보이게
- **배지**: 몇 개 획득 상태로 만들어 배지 하이라이트가 비어보이지 않게
- **소셜 피드**: 데모용 친구 계정 2~3개 만들어 서로 팔로우 + 각자 기록 2~3개 → 피드 스크롤이 채워지게
- **프리미엄**: 설정에서 isPremium 토글 ON (프리미엄 화면·혜택 촬영용)

### 세팅 체크리스트
- [ ] 데모 계정 로그인, 온보딩 완료(홈 국가 지정)
- [ ] 국가 8~12개 방문 처리 (지구본 색칠 확인)
- [ ] 기록 10~15개 (형식 골고루)
- [ ] 프로필 여행 카드 대표사진 다 지정
- [ ] 통계 화면 값이 그럴듯한지 확인 (24% 같은 %가 뜨는지)
- [ ] 데모 친구 계정으로 피드·댓글·좋아요 채우기
- [ ] 프리미엄 토글 ON

> 참고: `SHOW_VISITED_SHEET` 등 비활성 기능은 그대로 두세요(의도된 비활성). 아직 미완인 SocialExplore 탭은 촬영 대상에서 제외.
